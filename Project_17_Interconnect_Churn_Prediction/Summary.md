Sprint 17 | Interconnect Telecom Churn Prediction
Binary classification model to predict customer churn for a telecom operator using CatBoost, XGBoost, LightGBM, and Logistic Regression. Final model achieved AUC-ROC of 0.854 on held-out test data.
Tools: Python | Pandas | Scikit-learn | CatBoost | XGBoost | LightGBM | Matplotlib | Seaborn
